# Vite prototype (archived)

هذا المجلد يحتوي النسخة الأولية من الواجهة على **Vite + React** قبل اعتماد **`frontend/` (Next.js)** كمنتج رئيسي.

لا تُحدَّث هذه الحزمة كجزء من مسار الفحص الفني الحالي. للرجوع التاريخي أو استعادة مكوّن معيّن فقط.

لتشغيلها (اختياري):

```bash
cd legacy/vite-prototype
npm install
npm run dev
```
